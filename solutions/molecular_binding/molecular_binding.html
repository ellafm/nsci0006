Molecular binding#

Question 1#

Plotting the given function on the range \([0,10]\) :

#parameters
N=1.4; kd=0.05             #parameters given in question

c=np.linspace(0,10,1000)   #1000 points makes it smooth
Nb= N*c/(kd+c)

plt.plot(c,Nb)

#prettify (axis labels and ranges)
plt.xlabel('$c$ (nM)')
plt.ylabel('$N_b$ (pmol/mg)')
plt.xlim([0,10])
plt.ylim([0,1.45]);

Question 2#

Logspace produces a list of values that are equally spaced on a log scale. The syntax is logspace(a,b,npts) to create values between \(10^a\) and \(10^b\).

c=np.logspace(-4,1,1000)  # range [10^-4 to 10^1]
Nb= N*c/(kd+c)

plt.semilogx(c,Nb)        # display on a log scale (x axis)

# axis labels and ranges
plt.xlabel('$c$ (nM)')
plt.ylabel('$N_b$ (pmol/mg)')
plt.xlim([1e-4,10])

# add yticks and gridlines
plt.yticks([0,N/4,N/2,3*N/4,N])
plt.grid();
