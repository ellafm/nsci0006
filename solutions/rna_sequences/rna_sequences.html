Question 1#

The following expression extracts characters indexed 4-6 from the rna sequence. The indexing starts from 0, so this corresponds to letters 5-7

codon = rna_seq[4:7]
print(codon)

AUG

The following expression looks up the codon value AUG (the key) in the codon dictionary, and returns the corresponding amino acid, which is M

amino_acid = codon_dic[codon]
print(amino_acid)

M

Therefore the following equality test should return true

amino_acid=="M"

True

Question 2#

To find the position of the start codon:

n=len(rna_seq)   #number of characters in sequence

for i in range(0,n):
  codon=rna_seq[i:i+3]   #get the codon from the rna sequence
  if codon=="AUG":       #test the codon to see if it is AUG
    start=i              #record the start index
    break                #exit when position is found

print(start)

4

Question 3#

To convert to an amino acid sequence:

for i in range(start,n,3):        #start from AUG position in steps of three
  codon=rna_seq[i:i+3]            #identify the codon
  amino_acid=codon_dic[codon]     #use the dictionary
  if amino_acid=='.':             #stop if amino acid returns '.'
    break
  print(amino_acid)
