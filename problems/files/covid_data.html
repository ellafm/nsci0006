COVID data trends#

The coronavirus (COVID-19) in the UK dashboard is an up-to-date weekly government summary of key coronavirus metrics, such as levels of infection and hospitalisations. A download of the dashboard data to 22 Aug 2022 can be found in data_2022-Aug-17.csv.

The evolution of the infection can be characterised as a series of bell-shaped “waves”. It is tempting to use a normal (Gaussian) distribution to fit this type of data, but the logistic distribution that we will use in this exercise provides a better model.

Question 1#

Import the covid data and plot the number of daily cases in date order from oldest to newest.

Question 2#

The part of the dataset that we will focus on is from days 750-850, which roughly corresponds to a wave of infection in 2022 that occurred from mid February to the end of May. Extract this part of the dataset and label it y

Question 3#

Add the following line of code to remove low-level constant background cases - i.e. cases that are not associated with an infection wave:

y=y-min(y)

Plot the data together with the following function, taking \(r=0.1\), \(x_0=30\), \(E=2.5\times 10^6\) :

\[\begin{equation*} y=\frac{kE}{(1+E)^2}, \qquad E=e^{-r(x-x_0)} \end{equation*}\]

Read more

The function that you plotted above is the derivative of the logistic function. Conceptually, the logistic function models the cumulative (total) number of cases and its derivative models the changing (daily) number.

A plot of the total number of cases demonstrates the familiar S-shaped distribution as shown below. The vertical line on the plot has been added to mark the inflection point at \(x=30\). The data asymptotes towards the value \(y=2.5\times10^6\).
