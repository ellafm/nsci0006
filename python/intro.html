Introduction#

These notes provide a brief introduction to the programming language Python, to accompany the module NSCI0006. The content is split into nine chapters and it is intended that you should work through one chapter each week in your computing class. A tutor will be on-hand in the computer room to help you if you get stuck.

Completing the exercises#

The solutions for the exercises are provided in the form of drop-down boxes so that you can check your answers straight away. It is strongly recommended that you try these exercises yourself before peeking at the solution, as you will develop your understanding much faster this way.

What you will learn#

By the time you have read this book and completed the assessments, you should be able to:

  • Create an IPython notebook in CoCalc, and set out your work clearly using a combination of code and markdown.

  • Define variables and use these in calculations. Recognise the difference between variable types such as integers, floats, lists, arrays and dataframes.

  • Use single-element indexing to identify and manipulate items in an array or list. Use array slicing to extract more than one item from an array at the same time.

  • Produce plots either from a mathematical function definition or by importing a dataset. Fit straight lines and curves to data.

  • Apply an iterative method to generate a sequence of outputs, which may either be printed on-screen or stored in an array.

By working through the problems and attending the accompanying lectures you will also gain further understanding of mathematical ideas such as the connection between discrete (data) mathematics and continuous (calculus) mathematics.
