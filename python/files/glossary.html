Function glossary

Function glossary#

A non-exhaustive list of functions that are used in these notes is provided below. After working through this book you should be familiar with each of them and their use.


Native Python

The following functions are native to Python and do not require additional libraries to be installed

for [Section 2.2]
Carry out a series of repeated calculations, such as squaring every element in a list or implementing an iterative formula. The first line must end in a colon and the steps to be carried out in the loop must be indented.

range [Section 2.3]
Create a sequence of integers. The main use of this function is to control the steps that are carried out in a for loop.

def [Section 6.1]
Keyword to create a function definition. The first line must end in a colon and the steps to be carried out by the function must be indented. The final line should normally be a return statement that lists the outputs.


NumPy functions

The numpy library (pronounced “num-Pie”) allows us to define numeric arrays and to carry out numeric operations. The main functions that we use from this library are:

arange [Section 3.3]
Create a sequence of floats that are equally spaced between a defined start and end point. The step size between each two values must be specified.

linspace [Section 3.3]
Create a sequence of floats that are equally spaced between a defined start and end point. The number of values in the interval must be specified.

empty [Section 5.2]
Create an “empty” array of specified size. This is an important pre-allocation step if we are going to use an iterative approach to create the array. Note that ones and zeros can also be used for this purpose.

polyfit [Section 8.2]
Fit a polynomial (such as a straight line equation) to a given dataset. The output of the function is the fitted coefficients in descending power order.


Pandas functions

pandas is a powerful data analysis library. We barely use its functionality in this course, but we do borrow the simplicity and convenience that it allows when working with tabular data.

read_csv [Section 4.1]
This function can be used to import a csv file as a “dataframe”. The column headings can be used to extract columns from the dataframe, for instance to use as variables.


SciPy functions

The scipy library (pronounced “Sigh-Pie”) provides algorithms for optimization and scientific problem solving. We barely use its functionality in this course, but we do make use of its excellent curve fitting algorithm.

optimize.curve_fit [Section 8.3]
This function is used to fit a model to data. The model function must be defined in the manner model(x,coeffs) and the function returns the coeffs that provide the best fit in the sense defined by a least-squares algorithm.
