5. Iterative methods#

This chapter introduces key methodology that can be used to generate data sequences from a small number of inputs and governing rules. The implementaion draws on ideas and techniques that have been introduced in previous chapters such as variable editing and array indexing.

After completing this chapter you should be able to apply an iterative model to calculate values in a sequence and store them in an array.

5.1. Basic idea#

An iterative method produces a sequence of results in which the \(n\)th result is derived from the previous ones. The method requires one (or possibly more) initial values to be provided.

As an example, consider the following iterative formula to produce the factorial sequence \(F_n=n!\) based on the observation that \(n!=n(n-1)!\)

(5.1)#\[\begin{equation} F_0 = 1, \qquad F_{n}=n F_{n-1} \end{equation}\]

Explanation of the formula

The starting value \(F_0\) is the definition of \(0!\), which is required for mathematical consistency.

  • Using the result for \(F_0\) together with the iterative formula gives us the result \(F_1=(1\times F_0)=1\).

  • Using the result for \(F_1\) together with the iterative formula gives us the result for \(F_2=(2\times F_1)=2\).

Continuing in this manner allows us to sequentially compute the factorial values.

An implementation of the iterative formula in Python is shown below:

n=10       #Demonstration for 10!
F=1        #0!=1 by definition

for k in range(1,n+1):
  F=k*F                   #iterative formula

print(f"{k}! = {F}")        # see note below about f-strings

10! = 3628800

f-strings

Preceding a text-quote with the letter f signifies a special type of string, in which variable names enclosed in curly brackets {} are replaced with the value of the variable.

We used a similar approach to generate the Fibonacci sequence in the chapter on Lists and loops. In that case we started with two initial values and an iterative formula that gave each result as the sum of the two previous ones in the sequence.

Exercise 5.1

Use an iterative approach to print the next four results in the following sequence:

\[\begin{equation*} F_0=3, \qquad F_n=F_{n-1}+\sin(F_{n-1}) \end{equation*}\]

What do you observe?

Solution

F = 3

for k in range(1,5):
  F=F+np.sin(F)
  print(F)

The results appear to converge quickly towards the value \(\pi\).

5.2. Array creation#

In the examples that we have seen so far we simply printed the results on screen and allowed the variable values to be overwritten later in the calculation. However, there are many occasions where we would like to store the results of each iteration in a list or array. Doing so can also make it easier to implement the iterative formulae.

A neat feature of both lists and arrays is that their elements can be overwritten with new values. This allows us to begin with a “pre-allocated” list or array of blank or junk values and use a loop to fill in the values that we want.

In this course we will almost always work with arrays. We can use the empty function from the numpy library to create an array containing a specified number of elements. The values can then be filled in using indexing.

For example, to create a list of the first 10 factorial numbers:

n=10
F=np.empty(n)    #pre-allocate
F[0]=1           #0!=1 by definition        

for k in range(1,n):
  F[k]=k*F[k-1]       #iterative formula

print(F.astype(int))  #print values as integers

[     1      1      2      6     24    120    720   5040  40320 362880]

Exercises

Exercise 5.2

Generate and plot the first 50 value given by the iterative formula

\[\begin{equation*} x_{i+1} = x_i(1+r), \qquad x_0=1, \quad r=0.1 \end{equation*}\]

Can you recognise what type of growth is represented by this formula?

Solution

n=50                    #number of values
x=np.empty(n)           #pre-allocate
x[0]=1                  #first value
r=0.1                   #given parameter

for i in range(n-1):
  x[i+1]=x[i]*(1+r)     #iterative formula

plt.plot(x)
plt.xlabel('$i$')
plt.ylabel('$x_i$');

The formula represents exponential growth. The \(n\)th value of \(x\) is given by

\[\begin{equation*} x_n =x_0(1+r)^n. \end{equation*}\]

Exercise 5.3

Recall that the Fibonacci sequence is defined by the following iterative formula:

\[\begin{equation*} x_{i+1}=x_i+x_{i-1}, \qquad x_0=x_1=1 \end{equation*}\]

Implement the formula to create an array of the first 10 Fibonacci numbers.

Solution

n=10                     #number of values
x=np.empty(n)            #pre-allocate
x[0]=1; x[1]=1           #first two values

for i in range(1,n-1):
  x[i+1]=x[i]+x[i-1]     #iterative formula
print(x)

5.3. Coupled formulae#

Iterative methods may involve using the values of one variable to update another. For example, consider the model below in which there are two sets of values \(x_i\) and \(y_i\) that are co-dependent.

(5.2)#\[\begin{align} x_{i+1}&=x_i-0.5 y_i, \quad x_0=10\\ y_{i+1}&=y_i+0.4 x_i, \quad y_0=10 \end{align}\]

The code below shows how this model can be implemented in Python to generate the first 20 values for each variable. Notice that both iterative formulae can be included within the same loop.

n=20                    #number of values
x=np.empty(n)           #pre-allocate
y=np.empty(n)           #pre-allocate

x[0]=10; y[0]=10        #first values

for i in range(n-1):
  x[i+1]=x[i]-0.5*y[i]  #iterative formula
  y[i+1]=y[i]+0.4*x[i]

plt.plot(x,'r',label=r'$x_i$')
plt.plot(y,'b',label=r'$y_i$')
plt.xlabel('$i$')
plt.legend();
