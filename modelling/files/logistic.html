2. Logistic model#

The logistic model applies to scenarios where we anticipate exponential growth that levels off due to resource restrictions. It has been utilised in a tremendous range of scientific and non-scientific applications.

2.1. The model#

The model is given by the following equation, where \(x(t)\) is the quantity of interest, such as a population or stock:

(2.1)#\[\begin{equation} \frac{1}{x}\frac{\mathrm{d}x}{\mathrm{d}t}=\underbrace{r\left(1-\frac{x}{C}\right),}_{\text{relative growth rate}} \qquad x(0)=x_0. \end{equation}\]

Notice that the model is equivalent to the exponential model, but with a nonlinear relative growth rate. The model assumes that the relative growth rate is proportional to the fraction \(1-x/C\), where \(C\) is called the carrying capacity.

Note

The exponential model can be seen as a special case of the logistic model with infinite carrying capacity.

2.2. Production function#

According to the logistic model, the rate of change of stock \(x\) is given by the following production function:

(2.2)#\[\begin{equation} P(x)=rx\left(1-\frac{x}{C}\right). \end{equation}\]

Exercise 2.1

Sketch how the production function should look for postive parameter values \(r,C\). Describe how the growth rate changes in relation to the carrying capacity.

Solution

The production function is a quadratic, which may be written in completed square form:

\[\begin{equation*} P(x)=-\frac{r}{C}\biggr[\left(x-\frac{C}{2}\right)^2-\frac{C^2}{4}\biggr] \end{equation*}\]

We can see that the function has a maximum at \((C/2,rC/4)\). Therefore the model predicts that peak production occurs at half the carrying capacity. The growth rate (production) increases and then decreases symmetrically as a function of \(x\) until the carrying capacity is reached. A sketch of the production function is shown below:

According to the model, when the carrying capacity is reached there is no production and so the carrying capacity will not be exceeded.

2.3. Analytic solution#

The logistic model is satisfied by the following result, in which \(\tau\) is the time to peak:

(2.3)#\[\begin{equation} x=\frac{C}{1+e^{-r(t-\tau)}}, \qquad \tau=\frac{1}{r}\ln\left(\frac{C}{x_0}-1\right) \end{equation}\]

Note

Note: If \(g\) is discrete growth rate (e.g. annual growth rate) then \(r=\ln(1+g)\) and so we have

\[\begin{equation*} \tau=\frac{\ln(C/x_0-1)}{\ln(1+g)} \end{equation*}\]
